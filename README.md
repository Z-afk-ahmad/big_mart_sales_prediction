# 🛒 Big Mart Sales Prediction

This project focuses on predicting item outlet sales using Machine Learning techniques.

---

## 📊 Project Overview
The goal of this project is to analyze retail data and build a model that can predict the sales of products based on different features such as item type, outlet type, and more.

---

## 🔍 Steps Performed
- Data Cleaning
- Handling Missing Values (Mean & Mode)
- Encoding Categorical Variables
- Exploratory Data Analysis (EDA)
- Model Building

---

## 🧠 Model Used
- Linear Regression

---

## 📈 Results
The model was trained to predict sales and evaluate performance using standard regression metrics.
